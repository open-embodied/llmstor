# S.I.M.P

S.I.M.P (Simple Image Management Platform) is an open-source, self-hosted image sharing and analytics platform. It provides a RESTful API for authentication, image upload and management, analytics, and configuration, with a modern frontend and robust backend.

**Main Features:**

- JWT-based authentication
- Secure image upload and management
- Privacy controls for images
- Detailed analytics (views, countries, disk usage)
- Configurable via YAML
- Extensible and open source

**Use Cases**

- ShareX custom image uploader. See how to configure ShareX [here](/docs/sharex).
- Personal image/screenshot hosting
- Self-hosted alternative to Imgur or similar services
