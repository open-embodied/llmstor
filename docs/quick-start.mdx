## 1. Clone the Repository

```bash
git clone https://github.com/DanonekTM/SIMP.git && cd SIMP
```

## 2. Configure SIMP

Edit `backend/config.yaml` to match your environment. See the [Configuration Guide](/docs/configuration).

## 3. Deploy

Choose a deployment method:

- [Bare Metal](/docs/deployment/bare-metal)
- [Docker](/docs/deployment/docker)
- [Docker Compose](/docs/deployment/docker-compose)

## 4. Access the API

See the [API Reference](/docs/api) for endpoints and usage.
